# Cloud Data Platform

Projeto desenvolvido para estudos de Engenharia de Software, Engenharia de Dados e Computação em Nuvem (AWS), com foco na construção de uma plataforma de dados seguindo boas práticas de desenvolvimento profissional.

---

## Objetivo

Construir, de forma incremental, uma plataforma de dados moderna aplicando conceitos de:

- Engenharia de Software
- Engenharia de Dados
- Arquitetura de Software
- Cloud Computing (AWS)
- Qualidade de Código
- Testes Automatizados
- CI/CD

---

## Tecnologias

- Python
- Git
- GitHub
- PowerShell
- Visual Studio Code

> Novas tecnologias serão adicionadas ao longo do projeto.
